# Prerequisites

- Java 8+
- Gradle
- IntelliJ IDEA

# Setup

1. Clone the repository: `git clone https://github.com/moxaj/agent_game.git`
2. Import as an existing project (`Import project` \ `Gradle (with 'use default gradle wrapper')`)
